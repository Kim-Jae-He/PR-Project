# PR-Project

김제희의 개인포트폴리오 소스코드입니다.

# 📁포트폴리오 소개
* 로그인, 회원가입, 경력작성, 게시판, 댓글 기능들을 구현했습니다.

# 🗂️ 주요기능
## 로그인,회원가입
* 로그인 유효성검사
* 로그인 토큰발행
* 회원가입 유효성검사
* 회원가입 정규식사용

## 경력작성
* 오늘 날짜 이전 선택가능하도록 구현
* Material Icons 캘린더 사용
* 경력추가, 삭제, 수정

## 게시판,댓글
* 게시판목록
* 게시글 추가, 수정, 삭제
* 게시글 작성에 이미지 파일 첨부
* 최신순,오래된순,좋아요순,조회수순 나열
* 좋아요 기능
* 댓글작성, 댓글삭제, 댓글수정 구
